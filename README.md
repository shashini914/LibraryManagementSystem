A simple and interactive Library Management System built using C#.
This console application allows users to register members, buy books, borrow and return items, manage loans, and view details about members and their activity.

## Features

1. Register Members
   - Add new library members
   - Auto-assign membership types (A, B, C)
   - Track total number of items borrowed
  
2. Buy Books
   - Purchase new books and add them to the library inventory
  
3. Borrow Items
   - Borrow books/items
   - Auto-generate loan ID, loan date, and return date
   - Prevent borrowing if member exceeds loan limit
  
4. Extend Item
   - Extend the return date for an active loan
  
5. Return Items
   - Mark items as returned
   - Update member borrowing count
  
6. View Members
   - Shows all registered members

7. View All Items
   - Displays all books/items available in the library
  
8. View All Loans
   - Shows each loan with detailed timestamps
